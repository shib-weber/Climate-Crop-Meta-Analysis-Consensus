## 🌍 Overview

Climate change directly impacts crop yields worldwide. This project provides:

- **Collaborative analysis** of crop-climate datasets
- **Consensus-driven insights** using blockchain
- **Immutable logging** of contributions
- **Open-source framework** for researchers and stakeholders

The system combines **Python data analysis**, **smart contracts on Ethereum (Ganache)**, and a **web3 interface** to create a reliable and transparent ecosystem.

---

## ✨ Features

- **Blockchain-backed consensus:** All decisions are securely recorded on a private blockchain.
- **Smart contract management:** Deployable contracts that handle data submission and verification.
- **Meta-analysis aggregation:** Consolidates multiple datasets and provides consensus-based results.
- **Transparent and auditable:** Every data submission and analysis step is traceable.
